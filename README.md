# Smart Traffic Management System

## Overview

The Smart Traffic Management System is a simulation-based project developed in C that optimizes vehicle movement within a city road network. The system uses Data Structures and Algorithms (DSA) to calculate shortest routes, manage traffic congestion, prioritize emergency vehicles, and dynamically adjust traffic signals.

The project demonstrates real-world applications of graph algorithms, queue management, stack operations, and route optimization techniques used in modern intelligent transportation systems.

---
